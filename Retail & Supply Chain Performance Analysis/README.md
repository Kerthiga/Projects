PROJECT OVERVIEW 

This project focuses on analyzing retail sales and supply chain performance using a Power BI dashboard. It helps track important business metrics such as revenue, inventory levels, stockouts, and delivery performance.

The dashboard provides a clear and interactive view of how products perform, how efficiently inventory is managed, and how well deliveries are handled. It allows users to identify trends, compare performance across categories, and understand key operational challenges. Overall, the project demonstrates how data can be used to improve decision-making in retail and supply chain management.
